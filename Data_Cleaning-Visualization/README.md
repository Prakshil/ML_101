# Data Cleaning and Visualization

This folder contains notebooks focused on practical data cleaning, inspection, and interpretation workflows.

## Files

- `data_accessing_and_cleaning.ipynb`
  - Covers data loading, inspection, missing values, duplicates, and cleaning steps.
  - Now includes step-wise markdown explanations before code cells.
- `Data_interpretation.ipynb`
  - Covers interpreting cleaned data outputs and deriving insights.
  - Now includes concept explanations for each code section.
- `train.csv`
  - Sample dataset used by notebook examples.

## Recommended Learning Order

1. `data_accessing_and_cleaning.ipynb`
2. `Data_interpretation.ipynb`

## How To Use

1. Open a notebook in VS Code or Jupyter.
2. Read each markdown step before running the next code cell.
3. Execute cells from top to bottom for consistent results.
4. Compare outputs with the explanation in the inserted markdown blocks.

## Note

The notebooks were updated with instructional markdown cells (headings, concept, focus, and expected outcome) to improve readability and learning flow.
