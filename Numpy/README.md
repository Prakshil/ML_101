# NumPy Notebooks

This folder contains notebooks for learning NumPy from fundamentals to advanced usage.

## Files

- `numpy_fundamentals.ipynb`
  - Introduces arrays, indexing, slicing, shape handling, and core operations.
  - Includes step-wise markdown explanations before code cells.
- `numpy_advanced.ipynb`
  - Covers advanced array logic, reshaping, combination, and practical operations.
  - Includes concept-based markdown guidance for each code segment.

## Recommended Learning Order

1. `numpy_fundamentals.ipynb`
2. `numpy_advanced.ipynb`

## How To Use

1. Start with the markdown description before each code cell.
2. Run cells sequentially to preserve variable state.
3. Verify each output against the expected outcome note.
4. Re-run key sections after modifying input values to deepen understanding.

## Note

Additional markdown cells were inserted to explain the concept used in each code step with clear headings and outcomes.
