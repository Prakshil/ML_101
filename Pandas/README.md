# Pandas Notebooks

This folder contains notebooks for Pandas Series operations, grouping, and dataset merging techniques.

## Files

- `pandas_series.ipynb`
  - Covers Series creation, indexing, transformation, and related operations.
  - Includes detailed markdown explanations before code cells.
- `groupby.ipynb`
  - Demonstrates grouping, aggregation, and grouped analysis workflows.
  - Includes concept-focused markdown for each computation step.
- `merging.ipynb`
  - Demonstrates merge, join, concat, and table combination strategies.
  - Includes guidance markdown blocks for each code section.

## Recommended Learning Order

1. `pandas_series.ipynb`
2. `groupby.ipynb`
3. `merging.ipynb`

## How To Use

1. Read each inserted markdown block before running the associated code cell.
2. Execute cells top-to-bottom to keep dependencies valid.
3. Use outputs to verify your understanding of the stated concept.
4. Experiment with small code changes and observe differences.
